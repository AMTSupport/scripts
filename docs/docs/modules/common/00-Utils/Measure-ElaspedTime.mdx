## SYNOPSIS

## SYNTAX

```powershell
Measure-ElaspedTime [-ScriptBlock] <ScriptBlock> [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION

This function is used to measure the time it takes to execute a script block.

## EXAMPLES

### EXAMPLE 1

```powershell
Measure-ElapsedTime {
    Start-Sleep -Seconds 5;
}
```
