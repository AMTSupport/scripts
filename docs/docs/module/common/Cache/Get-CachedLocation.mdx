## SYNOPSIS

Get the location of a cached file, creating it if needed.

## SYNTAX

```powershell
Get-CachedLocation [-Name] <String> [[-MaxAge] <TimeSpan>] [[-IsValidBlock] <ScriptBlock>]
 [-CreateBlock] <ScriptBlock> [[-WriteBlock] <ScriptBlock>] [-NoCache] [-ProgressAction <ActionPreference>]
 [<CommonParameters>]
```

## DESCRIPTION

This function checks if the cache file exists and is valid.
If it doesn't exist, it calls the CreateBlock to generate the file.
The cache file is considered valid if it exists, is not older than MaxAge, and if provided, the IsValidBlock returns true.
