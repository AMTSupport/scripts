## SYNOPSIS

Writes a message to the console at the Error level if the ErrorPreference is not SilentlyContinue or Ignore.

## SYNTAX

### Splat (Default)

```powershell
Invoke-Error [-Message] <String> [[-UnicodePrefix] <String>] [-PassThru] [-Throw]
 [-ErrorCategory <ErrorCategory>] [-Caller <InvocationInfo>] [-CallerCmdlet <PSCmdlet>]
 [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

### InputObject

```powershell
Invoke-Error [[-InputObject] <Hashtable>] [-PassThru] [-Throw] [-ErrorCategory <ErrorCategory>]
 [-Caller <InvocationInfo>] [-CallerCmdlet <PSCmdlet>] [-ProgressAction <ActionPreference>]
 [<CommonParameters>]
```
