## SYNOPSIS

Writes a message to the console at the Information level if the InformationPreference is not Ignore.

## SYNTAX

### Splat (Default)

```powershell
Invoke-Info [-Message] <String> [[-UnicodePrefix] <String>] [-PassThru] [-ProgressAction <ActionPreference>]
 [<CommonParameters>]
```

### InputObject

```powershell
Invoke-Info [[-InputObject] <Hashtable>] [-PassThru] [-ProgressAction <ActionPreference>] [<CommonParameters>]
```
